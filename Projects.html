The Quiz Project

This project comprises five HTML pages in which the user is asked some travelling-related questions in order to determine his/her specific preferences.

This is a simple, fast and clear design with a background picture and radio buttons for each answer. The user is asked to answer five questions, by selecting one of the 5 possible answers. At the end of the quiz, they will receive a message detailing what type of traveller they are.

The Caesar Cipher

This is an application which is encodes and decodes a text message introduced by the user.

I chose to use a system with two keys for coding using values from 1 to 26. In order to encode, you have to choose the values for those two keys, select the Encoding radio button and then input the text message you want to encode. Later, you have to select the same numbers for key1 and key2 in order to decode the message back to the initial content.

The Text Adventure Game

An interesting adventure game in which the player needs to score 900 points in order to win.

During the game, the player/adventurer needs to answer questions, choose items or click an image.All these actions can add or remove points from the score visible at the bottom right corner of the game page.
