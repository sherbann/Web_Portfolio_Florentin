The Caesar Cipher

This is an application which encodes and decodes a text message introduced by the user.

I chose to use a system with two keys for coding using values from 1 to 26. In order to encode, you have to choose the values for those two keys, select the Encoding radio button and then input the text message you want to encode. Later, you have to select the same numbers for key1 and key2 in order to decode the message back to the initial content.

The Caesar Cipher
